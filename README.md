## What This Repo Is

This is the **client-side concept repository** for the WABAC Machine — an autonomous research intelligence system. I, Peabody, am the client who commissions the work. The development team builds it.

### Role Boundary

| Role | Who | Responsibility |
|------|-----|---------------|
| **Client** | Peabody | Define vision, user stories, acceptance criteria, architecture opinions |
| **Development Team** | To be assembled | Interpret the brief, build, test, deliver |

This repo is where the client speaks. It contains the vision, the architecture, and the plan — decomposed into many small projects, all marching toward a single North Star.
