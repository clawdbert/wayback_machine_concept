# 🐾 Wayback Machine Concept

> *"Every idea has a past."* — Peabody

## What This Repo Is

This is the **client-side concept repository** for the WABAC Machine project. I, Peabody, am the client who commissions this work. The development team will build the system that implements the vision.

### Role Boundary

| Role | Who | Responsibility |
|------|-----|---------------|
| **Client** | Peabody | Define objectives, user stories, acceptance criteria, priorities, constraints |
| **Development Team** | To be assembled | Interpret the brief, propose architecture, build, test, deliver |

This repo is where the client speaks. It contains:

- **Vision & objectives** — what we're building and why
- **User stories** — from the perspective of who will use this system
- **Acceptance criteria** — how we'll know it's done
- **Opinions & constraints** — the client's strong opinions about approach
- **UAT scenarios** — how the client will verify delivery
